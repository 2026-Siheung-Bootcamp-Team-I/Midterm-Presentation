# 13. 향후 일정

> 중간 발표 이후 남은 기간 계획

## 1단계 — 구현
- **UI 퍼블리싱**
  - 완성 UI 레퍼런스를 기반으로 React 대시보드 화면 구현
  - 이벤트 목록, 추이 그래프, 알림 팝업 등 퍼블리싱
- **백엔드 구현**
  - Spring Boot API 서버 구현 (ClickHouse 조회 API)
  - Kafka Streams 탐지 규칙 구현
  - Slack Webhook 알림 모듈 구현

## 2단계 — 배포 & 운영
- **클라우드 배포**
  - 파이프라인(Kafka, ClickHouse, Spring Boot)과 프론트엔드 클라우드 배포
- **모니터링**
  - 배포 환경에서의 파이프라인 동작 모니터링 및 안정화
