# 5. 완성 UI 레퍼런스

> 이 슬라이드에는 기존 EDR 서비스의 대시보드 화면 캡처(레퍼런스 이미지)를 배치

## 레퍼런스로 참고할 화면 구성

### ① 메인 대시보드
- 보안 점수 / 전체 상태 요약
- 실시간 이벤트 추이 그래프 (시간대별 탐지 건수)
- 최근 탐지된 이상 행동 목록

### ② 이벤트(알림) 상세 화면
- 탐지 시각, 발생 위치(호스트), 위협 유형
- 간략화된 요약 팝업 + AI 기반 조치 방법 제시

### ③ 로그 조회 화면
- ClickHouse에 저장된 원본 로그 검색·필터링
- 호스트별 / 시간별 조회

## 발표 포인트
- "기존 EDR 서비스들은 이런 화면 구성을 갖고 있고, 우리는 여기에 **보안 점수, AI 해결책, Slack 알림**을 더한 UI를 목표로 한다"
