# 4. 기능명세서

| 구분   | 기능              | 담당 기술               |
| ------ | ----------------- | ----------------------- |
| 수집   | PC·네트워크 행동 실시간 수집 | OSquery, Zeek    |
| 전달   | 수집 로그를 메시지 큐로 전달  | Kafka           |
| 탐지   | 저장 전 길목에서 위협 패턴 실시간 검사 | Kafka Streams |
| 저장   | 원본 로그 저장 + 빠른 집계   | ClickHouse       |
| 알림   | 위협 판정 시 Slack 즉시 발송 | Spring Boot     |
| 시각화 | 추이 그래프·로그 조회 대시보드 | React            |

> 위는 이번 구현 대상. 보안 점수 / AI 해결책 / 바이러스 역추적은 **향후 계획**
