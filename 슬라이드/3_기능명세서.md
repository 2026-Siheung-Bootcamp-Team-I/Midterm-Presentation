# 2. 기능명세서

| 구분   | 기능              | 설명                                                            | 담당 기술               |
| ------ | ----------------- | --------------------------------------------------------------- | ----------------------- |
| 수집   | 실시간 로그 수집  | PC 내부(RAM, CPU, 파일, 프로세스) 및 네트워크(DNS 등) 행동 수집 | OSquery, Zeek           |
| 수집   | 로그 전달         | 수집된 로그를 실시간으로 메시지 큐에 전달                       | Kafka                   |
| 탐지   | 실시간 이상 탐지  | 로그가 DB에 저장되기 전, 길목(메모리)에서 위협 패턴 실시간 검사 | Kafka Streams           |
| 탐지   | 윈도우 연산       | "최근 5초 동안 50번 이상 실패했는가?" 같은 시간 단위 집계       | Kafka Streams           |
| 저장   | 대용량 로그 저장  | 모든 원본 로그를 열(칼럼) 단위로 저장, 빠른 통계·집계          | ClickHouse              |
| 알림   | Slack 실시간 알림 | 위협 판정 시 Slack 채널로 즉시 메시지 발송 (이메일 포함)        | Spring Boot             |
| 시각화 | 실시간 대시보드   | 이벤트 추이 그래프, 로그 조회 화면 제공                         | React + Spring Boot API |
| 부가   | 보안 점수         | 시스템 보안 상태를 점수화하여 표시                              | -                       |
| 부가   | AI 해결책 제시    | 탐지된 이상 현상에 대한 조치 방법 안내                          | -                       |
| 부가   | 바이러스 역추적   | 이상 행동의 발생 경로 추적                                      | -                       |
