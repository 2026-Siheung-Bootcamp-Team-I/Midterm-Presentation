# 13. 현재 진행 상황

> 앞의 기술 설명 중 지금까지 실제로 구현한 부분과 남은 부분

## ✅ 구현 완료 (실제 동작 확인)

- **1 수집** — OSquery/Zeek → Kafka(events)로 로그가 실시간 수집되는 파이프라인 동작
- **2 탐지** — Kafka Streams 탐지 규칙 동작 (예: "5초에 50번 이상 실패" 포착)

> 파이프라인 **앞단(수집 → 탐지)** 이 실제로 돌아가는 것을 확인

## 🔨 남은 부분 (다음 단계)

- **3 알림** — Slack Webhook 연동
- **4 저장** — ClickHouse 적재 + 조회 API
- **5 시각화** — React 대시보드 화면

> 뒷단(알림 · 저장 · 시각화)을 이어붙여 전체 파이프라인을 완성하는 것이 다음 목표 → (14. 향후 일정)
