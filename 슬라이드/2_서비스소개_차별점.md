# 2. 서비스 소개 & 차별점

## 무엇이 다른가?

### 핵심 기능
- 컴퓨터 활동의 특성을 감지하고, **이상 현상을 찾아 조치를 제시**
- 분석 자료를 대시보드(사이트)에 표시하는 것을 넘어 다양한 부가기능 제공

### 차별점 (기존 EDR/백신 대비)
1. **보안 점수** — 현재 시스템의 보안 상태를 점수로 한눈에 표시
2. **AI 기반 해결책 제시** — 탐지된 이상 현상에 대한 조치 방법 안내
3. **간략화 팝업 표기** — 복잡한 로그 대신 이해하기 쉬운 요약 알림
4. **바이러스 역추적** — 이상 행동의 발생 경로 추적
5. **Slack 연동** — 바이러스·이상 행동 감지 시 사용자에게 즉시 전달

### AI 학습 기반 탐지
- AI가 **사용자의 평소 행동 양식**과 **컴퓨터의 평소 상태**를 학습·분석
- 평소 패턴에서 벗어난 **이상 행동 탐지에 용이**
